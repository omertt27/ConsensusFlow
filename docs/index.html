The Problem

The Trust Wall

Generative AI models produce confident, fluent, and wrong answers. Developers shipping AI products face a fundamental problem: they cannot verify the output of a single model without manually checking every fact.

This "Trust Wall" costs engineering time, creates liability risk, and destroys user confidence the moment one hallucination slips through.

❌
Single-model hallucinations GPT-4o alone hallucinates on ~30% of factual queries in production.
🚨
No audit trail Users get a final answer with no visibility into how it was derived.
💸
"Multi-model is 3× the cost" Running 3 models naively tripled token spend — until now.
The Solution

Sequential Verification

ConsensusFlow introduces a three-stage pipeline with an adversarial middle layer. Instead of asking models to agree, it forces them to disagree — finding friction before it reaches your users.

The Early Exit mechanism means you only pay the full chain cost when disagreement is detected. When all claims pass, the resolver is skipped entirely — saving up to 33% on tokens.

🔬
Atomic claim extraction Every fact is isolated and checked individually, not as a whole.
⚖️
Adversarial "Negative Reward" audit Auditors are penalised for agreeing too easily — they must find the flaw.
📄
Full Markdown audit trail Every correction, nuance, and verification is logged and exportable.

🚀 Quickstart

Up and running in 60 seconds

One install, one .env file, one line to verify anything.

$ pip install consensusflow

# Create your .env file
$ cat > .env << EOF
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
GEMINI_API_KEY=AIza...
EOF
import asyncio
from consensusflow import verify, render_markdown

# One line to verify any prompt
report = asyncio.run(verify(
    "Plan a 2-day trip to Istanbul",
    chain=[
        "gpt-4o",
        "gemini/gemini-2.5-flash",
        "claude-3-7-sonnet-20250219",
    ]
))

print(report.final_answer)
print(render_markdown(report))  # Full audit trail
# Basic verification
$ consensusflow "Is the Blue Mosque free?"

# Custom chain + save report
$ consensusflow "Topkapi Palace hours 2026" \
    --chain gpt-4o gemini/gemini-2.5-flash \
             claude-3-7-sonnet-20250219 \
    --output markdown \
    --save report.md

# Stream output live
$ consensusflow "Istanbul 2-day itinerary" --stream
import asyncio
from consensusflow import SequentialChain

async def main():
    chain = SequentialChain()

    # Yields events as each step completes
    async for event in chain.stream(
        "Plan a 2-day Istanbul trip"
    ):
        if event["event"] == "proposer_chunk":
            print(event["data"], end="", flush=True)
        elif event["event"] == "done":
            print("\n✅ Verified!")

asyncio.run(main())
